## 🚀 How to Compile (Local)

### latexmk (recommended)
```
latexmk -pdf -interaction=nonstopmode main.tex
```

### pdflatex (manual)
```
pdflatex main.tex
```

### Overleaf
1. Go to https://overleaf.com  
2. Create a new project  
3. Upload the `.tex` file
